# LEMP Application Hosting on AWS

## 📌 Project Overview
This project demonstrates how to host a modern LEMP stack application on AWS cloud infrastructure.

The application uses Linux, Nginx, MySQL, and PHP for high-performance web hosting and database connectivity.

---

## 🎯 Objective
- Host PHP web application using Nginx
- Configure modern LEMP architecture
- Connect application with MySQL database
- Deploy scalable web hosting infrastructure

---

## 🧰 AWS Services Used
- Amazon EC2
- Amazon RDS
- Nginx Web Server
- PHP
- MySQL

---

## 🏗 Architecture
User → Nginx Server (EC2) → MySQL Database (RDS)

---

## 🚀 Implementation Steps

### Step 1: Launched EC2 Instance
- Configured Linux server on AWS
- Connected to EC2 using SSH

### Step 2: Installed LEMP Stack
- Installed Nginx server
- Installed PHP and MySQL packages

### Step 3: Configured Database
- Created MySQL database
- Connected PHP application with database

### Step 4: Hosted PHP Application
- Uploaded PHP files to Nginx server
- Configured Nginx web hosting

### Step 5: Tested Application
- Verified Nginx server connection
- Checked browser output successfully
