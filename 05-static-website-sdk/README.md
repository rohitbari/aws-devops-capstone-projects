# Automate Static Website Hosting using boto3

## 📌 Project Overview
This project demonstrates how to automate static website hosting on Amazon S3 using Python boto3 scripts.

The Python script automatically creates an S3 bucket, enables static website hosting, configures bucket policies, and uploads website files.

---

## 🎯 Objective
- Automate static website deployment on AWS
- Upload website files to Amazon S3
- Enable static website hosting using Python automation
- Reduce manual AWS Console operations

---

## 🧰 AWS Services Used
- Amazon S3
- boto3 (AWS SDK for Python)

---

## 🏗 Architecture
Local Website Files → Python boto3 Script → Amazon S3 → Live Website

---

## 🚀 Implementation Steps

### Step 1: Configured AWS CLI
- Configured AWS credentials using `aws configure`

### Step 2: Created Website Files
- Created `index.html` and `error.html`

### Step 3: Developed Python Automation Script
- Created S3 bucket
- Configured public access policy
- Enabled static website hosting
- Uploaded website files automatically

### Step 4: Executed Deployment Script
- Ran Python script using boto3
- Generated live website URL

### Step 5: Tested Website
- Accessed deployed static website through browser
