# Bus Booking Application Deployment on AWS

## 📌 Project Overview
This project demonstrates the deployment of a real-world Bus Booking Application on AWS cloud infrastructure.

The application is hosted on Amazon EC2, connected with Amazon RDS for database management, and integrated with a Load Balancer for high availability and traffic distribution.

---

## 🎯 Objective
- Deploy a real-world web application on AWS
- Configure scalable cloud infrastructure
- Connect application with managed database
- Improve application availability using Load Balancer

---

## 🧰 AWS Services Used
- Amazon EC2
- Amazon RDS
- Application Load Balancer
- Security Groups

---

## 🏗 Architecture
User → Load Balancer → EC2 Application Server → Amazon RDS Database

---

## 🚀 Implementation Steps

### Step 1: Launched EC2 Instance
- Configured application server
- Installed required software and dependencies

### Step 2: Created Amazon RDS Database
- Configured relational database instance
- Connected application with database

### Step 3: Configured Load Balancer
- Created Application Load Balancer
- Attached EC2 instance to target group

### Step 4: Deployed Bus Booking Application
- Uploaded application files
- Configured application runtime environment

### Step 5: Tested Deployment
- Verified application accessibility
- Confirmed database connectivity
