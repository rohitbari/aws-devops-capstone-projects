# LAMP Application Hosting on AWS

## 📌 Project Overview
This project demonstrates how to host a traditional LAMP stack application on AWS cloud infrastructure.

The application uses Linux, Apache, MySQL, and PHP for web hosting and database connectivity.

---

## 🎯 Objective
- Host PHP web application on AWS
- Configure Apache web server
- Connect application with MySQL database
- Deploy traditional LAMP architecture

---

## 🧰 AWS Services Used
- Amazon EC2
- Amazon RDS
- Apache Web Server
- PHP
- MySQL

---

## 🏗 Architecture
User → Apache Server (EC2) → MySQL Database (RDS)

---

## 🚀 Implementation Steps

### Step 1: Launched EC2 Instance
- Configured Linux server on AWS
- Connected to EC2 using SSH

### Step 2: Installed LAMP Stack
- Installed Apache server
- Installed PHP and MySQL packages

### Step 3: Configured Database
- Created MySQL database
- Connected PHP application with database

### Step 4: Hosted PHP Application
- Uploaded PHP files to Apache server
- Configured Apache web hosting

### Step 5: Tested Application
- Verified Apache server connection
- Checked browser output successfully
